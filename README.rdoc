== README

* System dependencies
  - ruby 2.1 or later
  - rails 4
  - PosgreSQL

* Configuration
  - rake db:create
  - rake db:migrate
  - rake db:seed
  
  For TTD:
  - rake db:migrate RAILS_ENV=test
  - rake db:test:prepare
  - rake db:seed RAILS_ENV=test

* How to run:

Test Driven:
  - rspec spec
  
The Application:
  - Open a terminal.
  - Type rails s
  - Open a browser and type localhost:3000
